Como Criar
LottieFiles

3 formas — do mais simples ao mais profissional

O que é um Lottie? É uma animação vetorial em formato JSON, criada a partir do Adobe After Effects e exportada com o plugin Bodymovin. Roda em web, iOS e Android sem perda de qualidade, com arquivos minúsculos. O LottieFiles é a plataforma que centraliza biblioteca, editor e player desses arquivos.

🪄

A Forma Simples

Usar animações prontas do LottieFiles e embedar no site com 3 linhas de código.

Zero After Effects 5 minutos Sem código
1

Acesse lottiefiles.com

Vá em lottiefiles.com e busque a animação que quer. Há milhares grátis: loading, success, arrows, icons...

2

Baixe o arquivo .json ou copie o link

Clique na animação → Download → JSON. Ou copie o "Lottie URL" direto para usar via CDN.

3

Embed com o player oficial

Cole 1 script + 1 tag no seu HTML. Pronto. Sem dependências, sem build.

HTML — método mais rápido possível 
<!-- 1. Importa o player -->
<script src="https://unpkg.com/@lottiefiles/lottie-player@latest/dist/lottie-player.js"></script>

<!-- 2. Usa a tag (com URL ou .json local) -->
<lottie-player
  src="https://assets10.lottiefiles.com/packages/lf20_jcikwtux.json"
  background="transparent"
  speed="1"
  style="width:300px;height:300px"
  loop
  autoplay
></lottie-player>

A Forma Complicada

Escrever o JSON Lottie manualmente ou criar animações via código.

JSON à mão Horas Masoquismo
1

Entenda a estrutura do JSON Lottie

O Lottie é um JSON com especificação própria. Você precisa entender os campos principais antes de escrever qualquer coisa.

Estrutura mínima de um Lottie JSON 
{
  "v": "5.5.7",          // versão do Bodymovin
  "fr": 60,            // framerate
  "ip": 0,             // in-point (frame inicial)
  "op": 120,           // out-point (frame final)
  "w": 500,            // largura
  "h": 500,            // altura
  "nm": "My Animation", // nome
  "layers": [          // array de layers
    {
      "ty": 4,          // type: 4 = shape layer
      "nm": "Circle",
      "ip": 0,
      "op": 120,
      "shapes": [
        {
          "ty": "el",       // ellipse
          "p": { "v": [250, 250] },
          "s": { "v": [200, 200] }
        },
        {
          "ty": "fl",       // fill
          "c": { "v": [0, 0.9, 0.6, 1] } // RGBA 0-1
        }
      ],
      "ks": {            // keyframes de transform
        "s": {           // scale animado
          "a": 1,         // animated: true
          "k": [
            { "t": 0,   "s": [100, 100] },
            { "t": 60,  "s": [150, 150] },
            { "t": 120, "s": [100, 100] }
          ]
        }
      }
    }
  ]
}
2

Ou use a lib lottie-api / lottie-builder via Node

Algumas libs permitem construir o JSON programaticamente em JavaScript ou Python, sem After Effects.

Node.js — gerar JSON via código 
npm install lottie-builder

// animation.js
import { LottieBuilder } from 'lottie-builder';

const anim = new LottieBuilder({
  width: 500, height: 500, frameRate: 60, duration: 2
});

// Adiciona um círculo que pulsa
anim
  .addLayer('circle')
  .setColor('#00e5a0')
  .keyframe(0,   { scale: [100, 100] })
  .keyframe(60,  { scale: [130, 130] })
  .keyframe(120, { scale: [100, 100] });

fs.writeFileSync('output.json', anim.toJSON());
3

Valide com o LottieFiles Web Player

Arraste o .json em lottiefiles.com/preview ou use o Lottie Inspector para debugar keyframes e layers com problema.

4

⚠️ Por que é complicado

O formato Lottie não foi feito para ser escrito à mão. Curvas bezier, easing e transformações aninhadas ficam rapidamente ilegíveis. Só faça isso se tiver um motivo muito específico (ex: geração procedural de animações via código).

1

Instale o plugin Bodymovin no After Effects

Baixe via ZXP Installer ou direto em aescripts.com/bodymovin. É gratuito. Ativa em: AE → Window → Extensions → Bodymovin.

2

Boas práticas na criação da animação

Para o export funcionar bem, siga essas regras no After Effects:

Checklist AE para Lottie 
✓  Use vetores/shapes — nunca bitmaps/fotos
✓  Prefira Shape Layers ao invés de Illustrator layers
✓  Pre-comp só quando necessário (aumenta o .json)
✓  Use Easy Ease nos keyframes (exporta bezier correto)
✓  Evite efectos e plugins de terceiros (não exportam)
✓  Nomeie todos os layers (facilita controle via JS)
✓  Use Trim Paths para animações de linha/traço
✗  Nada de 3D layers, camera, rasterize contínuo
✗  Evite blending modes complexos (alguns não funcionam)
3

Export com Bodymovin

Abra o Bodymovin, selecione a composição, escolha a pasta de destino e clique em Render. Gera um data.json (+ assets se tiver imagens).

Configurações recomendadas no Bodymovin 
Settings recomendados:
  ✓ Glyphs         → converte fontes em paths
  ✓ Expressions    → interpola expressões simples
  ✓ Reduce Code    → minifica o JSON
  ~ Images         → embed como base64 (aumenta tamanho)
  ✗ Hidden Layers  → não exporta layers ocultos
4

Otimize com LottieFiles ou lottie-compress

Suba no LottieFiles para ver preview + métricas de performance. Use o lottie-compress para reduzir o tamanho do JSON.

Terminal — otimização 
npx lottie-compress input.json -o output.json

# ou com o CLI do LottieFiles
npx @lottiefiles/lottie-js compress data.json
5

Controle avançado em produção via lottie-web

Controle programático total: play, pause, speed, segmentos, eventos, controle frame a frame, interatividade via scroll.

JavaScript — controle profissional 
import lottie from 'lottie-web';

const anim = lottie.loadAnimation({
  container: document.getElementById('anim'),
  renderer: 'svg',     // 'svg' | 'canvas' | 'html'
  loop: false,
  autoplay: false,
  path: '/animations/data.json'
});

// Tocar só um segmento
anim.playSegments([0, 30], true);

// Controlar velocidade
anim.setSpeed(1.5);

// Ir para frame específico e pausar
anim.goToAndStop(45, true);

// Animação interativa com scroll
window.addEventListener('scroll', () => {
  const progress = window.scrollY / document.body.scrollHeight;
  const frame = Math.round(progress * anim.totalFrames);
  anim.goToAndStop(frame, true);
});

// Trocar cor de layer por nome
const api = lottie.loadAnimation({ ... });
api.addEventListener('DOMLoaded', () => {
  api.renderer.elements[0]
    .updateDocumentData({ fillColor: '#ff6b35' });
});

Alternativa moderna: Rive

Se você quer animações interativas com máquinas de estado (ex: hover → play → idle), considere o Rive como alternativa profissional ao Lottie. Mais controle, arquivo menor, mas curva de aprendizado diferente.
